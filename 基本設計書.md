## 全体構成図（API → 判定ロジック → 注文 → DB → ログ）
```mermaid
graph TB
    User((ユーザー))
    subgraph "全銘柄情報収集（日次）"
        DailyBuySellDecisionEngine[売買判定機能（日次）]
    end
    subgraph "保有資産銘柄情報収集（常稼働）"
        RealTimeBuySellDecisionEngine[売買判定機能（常稼働）]
        CoreTradingOperations[基本操作機能]
    end
User --> DailyBuySellDecisionEngine
User --> RealTimeBuySellDecisionEngine
RealTimeBuySellDecisionEngine --> CoreTradingOperations
```
